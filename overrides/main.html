v1.4.0.0

Welcome to IVyTools v1.4 🎉

This release brings key enhancements for orthorectification and STI review to give you more control and clarity. Here's what's new:

  • Enhanced STI Review: Manual correction lines now appear as dotted red lines alongside averaged and auto-detected lines, plus a new "Clear Manual Corrections" button for quick resets.
  • Pixel GSD Override: Manually set a known Ground Sample Distance when computed values aren't reliable—ideal for precision workflows.
  • Full-Motion Video GCP creation: TS (Transport Stream) Conversion Tool now correctly computes the ground control information for supported UAS nadir videos based on video metadata and a user-specified Height Above Water (HAW).

Tip: Projects from v1.x should load fine, but reprocessing is recommended to take advantage of these improvements.

Want the full details? Check out the v1.4.0.0 Release Notes.

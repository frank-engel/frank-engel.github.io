v1.5.0.0

Welcome to IVyTools v1.5 🎉

This is a major update bringing interactive lens correction, smarter search line handling, improved velocity visualization, and friendlier error reporting. Here's the highlights:

  • Interactive Lens Distortion Correction: A new dialog lets you preview and adjust radial distortion coefficients (k1, k2) with real-time visual feedback before processing. GCP digitization now prompts you to apply corrections automatically.
  • Search Line Anchor Modes: Control where the grid node sits along the search line (Start, End, Center, or a custom Percentage) giving you flexibility for different field setups.
  • Automatic Search Line Clipping: Lines extending beyond the image boundary are now clipped automatically, eliminating the NaN velocities that previously occurred at velocity nodes where the search line "ran off the page."
  • Vector Scale & Reference Vector: Adjust velocity arrow lengths on the fly with a new spinbox, and toggle a reference vector legend for quick visual interpretation.
  • STIV Image Export: Export the current STIV view—including all vectors—to JPEG or PNG via Shift + right-click.
  • Human-Readable Error Dialogs: Errors now display in clear dialog boxes with context and suggested actions instead of raw code tracebacks.

Plus: improved orthorectification for portrait/oblique cameras, add/remove GCP rows directly in the table, documentation migrated to MkDocs, and numerous bug fixes for units, uncertainty, and project loading.

Tip: Existing v1.x projects will load normally. Reprocessing is recommended to benefit from the search line clipping and ortho improvements. Older projects using 3D rectification will need to be review, as the new 3D method will be slightly different (XS will move).

Full details in the v1.5.0.0 Release Notes.
